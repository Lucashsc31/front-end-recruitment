## Stack
- Angular 6
  - Rxjs (State)
  - TypeScript
- Nodejs
  - ExpressJS
- NPM
- LocalStorage
- CSS
  - BEM
  - SCSS
- Icons
  - Font Awesome

## How to build/run

At the project root run the following commands.

To install the packages:
```
npm install
```

To start the App (NodeJS and Angular):
```
npm start
```


Open the App in http://localhost:4200
